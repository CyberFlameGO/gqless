> See also, [limitations of getAccessor](/gqless/getAccessor#limitations)

Updates the value for a given accessor, by performing a pattern match against all other values of same type. See [Matchers](/matchers) for more.

## Example

Set `query.me` to a User in the cache, with a name of `bob`:

```ts
import { matchUpdate } from 'gqless'

const updated = matchUpdate(query.me, { name: 'bob' })

if (!updated) {
  throw new Error('bad match')
}
```
