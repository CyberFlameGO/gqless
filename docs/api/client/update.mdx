title: update

> See also, [limitations of getAccessor](/api/gqless/getAccessor#limitations)

Updates the data for a given accessor, performing a merge-update (same as when fetched from the network).

## Example

```ts
import { update } from 'gqless'

// Update scalars
update(query.me.name, 'bob')

// Merge in new data (existing fields left untouched)
update(query, { me: { name: 'bob' } })
```
