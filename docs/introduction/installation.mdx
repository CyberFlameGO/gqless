title: Installation
---

To install the runtime client and the codegen CLI:

```bash yarn2npm
yarn add gqless
yarn add -D @gqless/cli
```

## Subscriptions support

To add [subscriptions](https://github.com/graphql/graphql-spec/blob/main/rfcs/Subscriptions.md) support:

```bash yarn2npm
yarn add @gqless/subscriptions
```

## React bindings

To use `gqless` with [React](https://reactjs.org/)

```bash yarn2npm
yarn add @gqless/react
```

## Development tools

To improve development experience, you can add a logger:

```bash yarn2npm
yarn add @gqless/logger
```
